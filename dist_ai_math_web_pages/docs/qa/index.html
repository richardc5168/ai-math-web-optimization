外部 QA（提示優化流程）

目標：把所有題型的題目/答案/提示匯出 → 交給外部模型做 QA → 產生可人工確認的 patch → 逐步把提示變得更清楚（小五能懂）。

流程（一步一步）

匯出 外部模型 回填
  1. 匯出所有題型：scripts/export_all_questions.py
  2. 把 JSONL 丟給外部模型，要求只輸出 review JSONL
  3. 驗證回饋檔：scripts/validate_question_reviews.py
  4. 產生 patch：scripts/apply_question_reviews.py
  5. 人工確認後套用 patch，並把要啟用的 override 設為 approved=True
回到主頁
