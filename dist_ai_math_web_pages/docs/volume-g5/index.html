（按「出新題」開始）
提示：先想單位（cm³ / m³），再用公式計算；複合形體要「分解」再相加。

重點整理（不抄原題）

1) 體積單位：常用 cm³、m³（「立方」表示三個方向都算）。
2) 正方體體積：V = 邊長³
3) 長方體體積：V = 長×寬×高
4) 也可寫成：V = 底面積×高
5) 換算：1 m = 100 cm，所以 1 m³ = 100³ cm³ = 1,000,000 cm³
6) 複合形體：畫輔助線分解成幾個長方體/正方體，分別算體積再相加。
