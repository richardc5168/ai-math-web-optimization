依孩子目前想法，給「下一步」
你可以輸入孩子目前的想法（例如："我想先通分"、"我先約分"、"我想先乘再約分"）。 這頁會用純規則給一個「下一步」，不直接爆出最後答案。

常用解題流程

約分：分子分母同除以最大公因數。
同分母加減：分母不變，分子相加/相減，最後再約分。
異分母加減：先找最小公倍數通分，再做同分母加減。
乘法：能先約分就先約分（交叉約分），再分子乘分子、分母乘分母。
帶分數互換： a b/c = (a×c + b)/c； n/d = (n÷d) ...
