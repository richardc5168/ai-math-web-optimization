依孩子目前想法，給「下一步」
你可以輸入孩子目前的想法（例如：「我先把小數點去掉」、「我覺得要補 0」、「我想用估算」）。 這頁會用純規則回一個「下一步」，不直接爆出最後答案。

小數不失誤三招

1) 方向檢查：乘 0.x 會變小；乘 >1 會變大；除以 >1 會變小。
2) 整數先算：先去掉小數點算整數，最後放回小數點。
3) 估算範圍：先四捨五入算大概，答案不要差太多。
